# greenai-content
## リポジトリ内フォルダ構造

- `/topics`：毎朝9:00にマーケットニュースを収集し、選択したトピックに関するMarkdownファイルを1日1件格納します。
- `/articles`：`/topics` に記載されたトピックをもとに、Green AIに関連づけて執筆した記事を1日1件格納します。

## 処理の流れ

以下の処理を、毎朝9:00にスケジュールされたClaudeベースの仕組みで全自動実行しています。

1. 毎朝9:00に最新のマーケットニュースを収集
2. 以下のカテゴリから対象を選択
   - A. 制度・規制トレンド
   - B. 大企業バイヤー要求トレンド
   - C. エネルギー価格・電力需給トレンド
   - D. 海外エネルギー領域動向トレンド
   - E. 補助金・投資判断トレンド
   - F. 現場実装トレンド
   - G. 技術・設備トレンド
   - H. 開示・評価基準トレンド
3. 収集したニュースの中から、Green AIとの関連性が高いトピックを1つ選定
4. 直近で重複しないようにトピックを選定
5. 選定したトピックを `/topics` にpush
6. `/topics` の内容をもとに、`/articles` 用の記事を作成
7. 作成した記事を `/articles` にpush
