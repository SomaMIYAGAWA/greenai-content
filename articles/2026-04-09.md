# 「調達アンケートにCO2欄が追加された」——大企業バイヤーがScope3一次データ収集を年次義務化、2次・3次サプライヤーへの波及が加速する2026年

**メタディスクリプション：**
2027年3月期のScope3開示義務化を前に、大企業バイヤーが取引先へのCO2排出量報告を年次調達プロセスに正式組み込み始めた。1次から2次・3次サプライヤーへ波及する「データ収集連鎖」の全体像と、製造業が今年度中に整えるべき実務対応を解説する。（150字）

---

取引先評価アンケートに「CO2排出量（Scope1・Scope2）」の記入欄が突然追加された——そんな連絡がバイヤーから届いたとき、あなたの会社はすぐ回答できますか。製造業では、こうした「脱炭素調達アンケート」が2026年度から本格的に広がっており、データを出せないサプライヤーには取引条件の見直しが迫られる現実が静かに近づいている。

---

## 1. なぜ「今年度」に動くのか：SSBJ任意適用開始とScope3開示2027年義務化の構造

2025年3月、サステナビリティ基準委員会（SSBJ）が日本版ISSB基準（サステナビリティ開示基準）を正式公表した。適用スケジュールは以下のとおりだ。

| 時期 | 対象企業 |
|---|---|
| 2026年3月期〜 | 任意適用開始（すべての上場企業） |
| 2027年3月期〜 | 義務化：時価総額3兆円以上 |
| 2028年3月期〜 | 義務化：時価総額1兆円以上 |
| 2029年3月期〜 | 義務化：時価総額5,000億円以上 |

SSBJ基準では、気候関連情報としてScope1・Scope2に加え**Scope3（バリューチェーン全体のGHG排出量）**の開示が求められる。製造業においてScope3がGHG総排出量に占める割合は**8割超**に及ぶとされており、大企業がScope3を精度高く開示するには、サプライヤーから**一次データ（実測値ベース）**を収集することが不可欠だ。

日本経済新聞も「CO2開示基準、供給網分も対象 日本国内で2026年度以降」と報じており、義務化の1〜2年前にあたる2026年は、大企業にとって**サプライヤーへのデータ収集体制を整備する「準備の年」**となっている。

## 2. 大企業バイヤーはどう動いているか：調達アンケートへのCO2項目追加と年次KPI化

従来のサプライヤー評価は「品質・コスト・納期（QCD）」が中心だった。しかし2026年度から多くの大企業が、これに**「脱炭素（サステナビリティ）」**を第4の評価軸として加えつつある。具体的な動きとして以下が報告されている。

- **年次調達アンケートへのGHG項目追加**：取引額上位サプライヤーへの年1回の定期調査にScope1・Scope2排出量の実績値記入欄を設ける
- **削減目標・ロードマップの提出要請**：「2030年目標」や「脱炭素ロードマップの有無」を評価項目に組み込む
- **CO2排出原単位（製品・工程別）の提供依頼**：Scope3カテゴリ1（調達原材料）算定に必要な「単位製品あたりのCO2量」をサプライヤーに求める
- **サプライヤーポータルの整備**：日立やNECなど大手は専用プラットフォームでのデータ収集を開始。回答しなければ評価対象外となる仕組みを構築

「今はまだお願いベース」という状況は急速に変わりつつある。SBTi（Science Based Targets initiative）の認定取得や再エネ電力100%化を取引条件に明記する企業も欧米を中心に増えており、日本企業も遅れて追随する動きが顕在化している。

## 3. 要求連鎖のメカニズム：1次→2次→3次サプライヤーへ波及する「データ収集ドミノ」

バイヤー要求が1次サプライヤーにとどまらない理由は、**Scope3の算定構造**にある。

```
【大企業（SSBJ義務化対象）】
  └─ Scope3カテゴリ1（調達品・原材料）を算定するには
       └─ 1次サプライヤーの製品・サービスごとのCO2排出量が必要
            └─ 1次サプライヤーのScope3カテゴリ1を算定するには
                 └─ 2次サプライヤーのCO2排出量が必要
                      └─ 以下、3次・4次へ連鎖
```

GHGプロトコルの「Scope3算定の段階的アプローチ」では、**取引額上位80%を占めるサプライヤー**を特定し、その中で炭素集約度の高い企業から優先的に一次データを収集することが推奨されている。これは1次サプライヤーが、自身の大企業顧客から求められるデータ精度を上げるために、さらに2次・3次サプライヤーに同様のアンケートを送らざるを得ない構造を生む。

**結果として：**
- 大手自動車メーカーからの要求 → 部品メーカー（1次）へ → 素材・金属加工業者（2次）へ → 原材料供給企業（3次）へ
- 大手電機メーカーからの要求 → 電子部品メーカー（1次）へ → 化学品・基板メーカー（2次）へ

2026年度以降、この「連鎖」は業種横断で加速する見通しだ。

## 4. 中堅・中小サプライヤーが直面する実務課題：何を・どの単位で・いつまでに用意するか

バイヤーからのアンケートに回答するには、少なくとも以下の情報を自社で把握している必要がある。

### 最低限求められるデータ項目

| 項目 | 内容 |
|---|---|
| Scope1排出量 | 自社工場での燃料（重油・ガス・石炭）燃焼によるCO2排出量（tCO2） |
| Scope2排出量 | 購入電力によるCO2排出量（マーケット基準・ロケーション基準） |
| 排出量の算定根拠 | 使用した排出係数・算定基準（GHGプロトコル等） |
| 前年比削減率 | 年度ごとの排出量推移 |
| 削減目標・ロードマップ | 2030年・2050年目標の有無 |

### 多拠点企業ならではの困難

- 拠点ごとにエネルギー請求書・燃料使用実績が分散
- 本社の環境部門に集計情報が届くまでに数ヶ月のタイムラグ
- バイヤーごとにフォーマットが異なり、転記作業が膨大
- 複数のバイヤーから同時期にアンケートが届き、対応が追いつかない

ある多拠点の部品メーカーでは「3つのバイヤーから異なるフォーマットでアンケートが届き、担当者が2週間かけても回答できなかった」という事例もある。データ管理体制なしの「手集め・手転記」での対応には限界がある。
