# EU-CBAM本格適用で「工場別CO2排出量データ」が輸出条件に——鉄鋼・アルミ製造業が今すぐ整備すべき排出量管理の3ステップ

**メタディスクリプション：** EUの炭素国境調整措置（CBAM）が2026年から本格適用へ。鉄鋼・アルミ等の輸出製造業は工場別CO2排出量の実測データ提供が事実上義務化。対応ステップと管理体制構築のポイントを解説します。（157文字）

---

「御社の工場ごとのCO2排出量を、製品1トンあたりで今すぐ出せますか？」——欧州のバイヤーからこんな要請が届いた日本の製造業担当者が急増している。EUの炭素国境調整措置（CBAM）が2026年1月から本格適用フェーズに入り、鉄鋼・アルミニウム・セメントなどをEUへ輸出する日本メーカーは、「排出量データを出せなければ取引コストが増大する」という現実に直面しつつある。これはもはや「将来の規制リスク」ではなく、「今期の輸出条件」の話だ。

---

## 1. CBAMとは何か——2026年から「本番」が始まった欧州の炭素国境税

CBAM（Carbon Border Adjustment Mechanism：炭素国境調整措置）とは、EU域外から輸入する製品に対してEU-ETS（排出量取引制度）と同等の炭素コストを課す仕組みだ。「EU域内の工場が炭素コストを負担しているのに、規制の緩い国からの輸入品が安く入ってくるのはフェアではない」という「炭素リーケージ防止」の観点から設計された制度である。

| フェーズ | 期間 | 主な内容 |
|---|---|---|
| 移行期間 | 2023年10月〜2025年12月 | CBAM報告書（四半期）の提出のみ義務化。実際の費用負担なし |
| **本格適用開始** | **2026年1月〜** | **CBAM証書の購入義務が発生。炭素コストの実費負担が始まる** |
| 拡大フェーズ | 2027年以降 | 川下製品（鉄鋼・アルミ使用製品等）約180品目への対象拡大予定 |

2026年からは、EU側の輸入業者が対象品目の「内包排出量（Embedded Emissions）」に応じてCBAM証書を購入しなければならない。証書の価格はEU-ETS炭素価格に連動するため、炭素集約度の高い製品ほど証書費用が増大する仕組みだ。

現在のEU-ETS炭素価格は1トンCO2あたり50〜70ユーロ前後（2026年時点）で推移している。仮に鉄鋼1トンに2tCO2が内包されているとすれば、それだけで100〜140ユーロ（約1.6万〜2.2万円）の証書費用が生じる。これがCBAM証書として輸入業者の実費コストとなるため、EU側バイヤーは「CO2排出量の少ない製品を優先調達したい」というインセンティブを持つ。

## 2. 移行期間終了で何が変わったか——「デフォルト値」から「実測値」へのシフト

CBAM対応で多くの日本企業が直面している本質的な課題が、**排出量データの算定方法の変化**だ。

移行期間中（〜2025年12月）は、EUが準備したデフォルト値（各品目の標準排出強度）を使えば、実際の排出量を計測・提出しなくても報告義務を果たせた。しかし、2024年第3四半期（2024年7月〜）報告以降、このデフォルト値の無制限使用は認められなくなった。

**デフォルト値を使い続けるリスク：**
- デフォルト値はEU域内の「最悪ケースの排出強度」をベースに保守的に設定されている
- つまり、実際より大幅に多い排出量と見なされ、CBAM証書費用がかさむ
- EU輸入業者はそのコスト増を嫌い、「実測データを提供できないサプライヤーとの取引を減らす・切り替える」行動が現実に起き始めている

**日本側輸出企業が提供すべき実測データの形式（EU規則準拠）：**

| データ項目 | 内容 |
|---|---|
| 直接排出量（Scope1相当） | 製造プロセスで直接排出されるCO2量 |
| 間接排出量 | 使用した電力由来のCO2排出量 |
| 算定単位 | 製品1トンあたり（tCO2/t）で提出 |
| 集計単位 | 施設（工場・拠点）ごとの個別データ |
| 証跡 | 算定根拠（エネルギー請求書、排出係数出典等）の保存・提出 |

## 3. 日本製造業への具体的影響——対象品目・輸出規模とコスト増リスク

ジェトロの試算によれば、日本からEUへのCBAM対象品目の輸出額は年間約30億ユーロ（約4,500億円）規模。鉄鋼・アルミニウムが中心を占め、自動車部品や産業機械を通じた間接的な波及まで含めると、影響範囲はさらに広がる。

**主な対象品目と日本企業への影響度（2026年現在）：**

| 品目 | CBAM対象 | 日本企業への影響度 |
|---|---|---|
| 鉄鋼・鋼材 | ○ | 高（大手・中堅ともにEU輸出実績大） |
| アルミニウム | ○ | 高（自動車部品向けが多い） |
| セメント | ○ | 中（専業輸出は限定的） |
| 肥料 | ○ | 低 |
| 電力・水素 | ○ | 限定的（輸出形態が特殊） |
| **川下製品（2027年〜）** | **拡大予定** | **高（鉄鋼・アルミ使用製品全般に波及）** |

さらに欧州委員会は2025年にCBAM強化策を追加発表し、2027年以降は川下製品（自動車部品・建材・産業機械部品など）への拡大を予告している。現時点でCBAMの「圏外」にいる部品メーカーも、数年以内に対応を迫られる可能性が高い。

**EU-ETS炭素価格60ユーロ（約9,300円）を前提としたCBAM証書費用試算：**

| 品目 | 平均排出強度（tCO2/製品t） | CBAM証書費用（円/製品t） |
|---|---|---|
| 鉄鋼（高炉） | 約1.8〜2.1 | 約1.7万〜2.0万円/t |
| アルミニウム（一次） | 約8〜12 | 約7.4万〜11.2万円/t |
| セメント | 約0.7〜0.9 | 約0.6万〜0.8万円/t |

鉄鋼でも1トンあたり2万円弱の証書費用が発生する計算であり、年間数万トンの輸出規模があれば証書費用の総額は数十億円に達する。これをEU輸入業者が全額負担するのか、価格転嫁を求めてくるのかは商流によって異なるが、いずれにせよ「自社のCO2排出量が少ないことを実測値で示せる企業」が価格競争上の優位を持つことは明白だ。

## 4. 施設別排出量データをどう準備するか——実務担当者のための対応ステップ

CBAM対応で最大の障壁となるのは「工場・拠点ごとのCO2排出量を、製品1トンあたりの数値で継続的に算出・証跡管理できる体制の構築」だ。多拠点を持つ製造業では特に、以下の課題が頻発している。

**よくある現場の課題：**
- 拠点ごとにエネルギー使用量の集計ルール・フォーマットが統一されていない
- 複数製品を同一工場で製造しているため、製品別の排出量按分が困難
- 電力排出係数の年次更新が手動で、担当者が異動すると管理が途絶える
- データの根拠（証跡）が残っておらず、EU側の監査・検証要求に応じられない

**CBAM対応の3ステップ：**

**STEP 1：現状把握と対象拠点の特定**
どの製品をどの工場で製造してEUに輸出しているかを品目・拠点レベルでリストアップする。CBAM対象品目に関わる工場を「優先拠点」として選定し、データ整備の順序を決める。

**STEP 2：拠点別・製品別の排出量算定体制を構築する**
優先拠点について、Scope1（燃料燃焼）・Scope2（電力）の排出量を月次で算定できるルールとフォーマットを確立する。複数製品を製造する工場では、エネルギー使用量を製品別に按分する方法論（生産量比、稼働時間比など）を文書化し、社内で統一する。

**STEP 3：証跡管理と継続的な運用体制を整備する**
算定根拠（エネルギー請求書、使用した排出係数の出典、按分計算シートなど）を一元的に保管し、EU輸入業者や監査機関からの照会に即座に対応できる状態を維持する。四半期ごとにデータを更新し、EU側に提供するフローを社内プロセスとして定着させる。

## 5. 2027年以降の拡大を見据えた「先手管理」の重要性

欧州委員会の公表内容によれば、CBAMは2027年以降に川下製品（鉄鋼・アルミを素材として使用する製造品）約180品目へ拡大される予定だ。これにより、これまで「素材メーカーの問題」だったCBAMが、**自動車部品・建材・産業機械部品を欧州向けに輸出する組み立てメーカーにも直接影響する**。

さらに、EUのCBAMに追随する形で英国・カナダ・オーストラリア等が類似制度の検討・導入を進めており、「欧州だけの規制」から「グローバルな輸出条件の変化」として捉え直す必要がある局面に入りつつある。

2027年の拡大を見据えると、今から始めておくべきことがある。対象が拡大した段階から慌てて体制整備を始めると、最初の数四半期はデフォルト値の適用を余儀なくされ、証書費用の過大負担・EU取引先からの評価低下というダブルのリスクを負う。逆に、今CBAM対象外でも「工場別CO2排出量の管理体制」を構築しておけば、拡大時に即座に対応でき、取引先に対して「脱炭素管理が進んでいるサプライヤー」としての差別化につながる。
