# 【2027年3月期義務化】SSBJ基準確定でサプライヤー製造業に迫る「Scope3データ提供要請」——今すぐ備えるべきGHG排出量管理の実務ポイント

**メタディスクリプション：**
2025年3月に確定したSSBJ基準により、2027年3月期から上場大企業のScope3開示が義務化。サプライヤーである製造業中堅・中小企業にもGHG排出量データの提供要請が急増。義務化対象外でも「間接義務化」への備えが急務です。

---

「うちは非上場だから、開示義務なんて関係ない」——そう思っている製造業の担当者ほど、いま注意が必要です。2025年3月、日本版サステナビリティ開示基準（SSBJ基準）が正式に確定し、2027年3月期からプライム市場上場企業へのScope3（サプライチェーン排出量）開示義務化が始まります。大企業がScope3を開示するためには、取引先である製造業サプライヤーのGHGデータが不可欠。義務化の波は、直接の適用対象ではない中堅・中小企業にも静かに、しかし確実に押し寄せています。

---

## 1. SSBJ基準とは何か——2025年3月確定、義務化タイムラインを整理する

SSBJ（サステナビリティ基準委員会）は、国際サステナビリティ基準審議会（ISSB）が策定したIFRS S1・S2をベースに、日本企業向けの独自開示基準を策定する機関です。2025年3月5日、長期にわたるパブリックコンサルテーションを経て、「SSBJ基準」が正式に確定・公表されました。

### 義務化の段階的スケジュール

| 適用時期 | 対象企業 | 主な開示要件 |
|---|---|---|
| 2026年3月期〜 | 任意適用（全上場企業） | Scope1・2（+Scope3は任意） |
| 2027年3月期〜 | プライム市場（時価総額3兆円以上） | Scope1・2・3 義務 |
| 2028年3月期〜 | プライム市場（1兆円以上） | 同上 |
| 2029年3月期〜 | プライム市場（5,000億円以上） | 同上 |
| 2030年代〜 | プライム市場全体 | 同上 |

現在（2026年4月）は任意適用フェーズの開始直後。この1年間は大企業がデータ収集体制を整備する「準備期間」であり、**サプライヤーへの問い合わせ・要請が本格化するタイミング**でもあります。

SSBJ基準はGHGプロトコルに準拠した排出量算定を前提としており、Scope1（自社の直接排出）・Scope2（購入エネルギー由来の間接排出）はもちろん、**Scope3（サプライチェーン全体の間接排出）の開示が義務の中核**となります。Scope3の15カテゴリのうち、製造業においては「購入した製品・サービス（カテゴリ1）」「輸送・配送（カテゴリ4）」などが特に重要で、これらは仕入先・外注先のデータなしには算定できません。

## 2. なぜ製造業サプライヤーに「間接義務」が生じるのか——Scope3開示の仕組みを理解する

Scope3開示の本質は「バリューチェーン全体の排出量を把握・報告する」ことにあります。トヨタ自動車やパナソニックなどの大手メーカーがSSBJ基準に基づいてScope3を開示するためには、部品・素材・加工を担うサプライヤーの排出量データが必要不可欠です。

### データ連鎖の構造

```
大手メーカー（SSBJ義務化対象）
  → Scope3カテゴリ1「購入した製品・サービス」を算定するために
  → 1次サプライヤー（中堅製造業）にGHGデータを要請
      → 1次サプライヤーが自社のScope3を算定するために
      → 2次サプライヤー（中小製造業・部品加工）にもデータを要請
```

この「Scope3データ連鎖」が実際に動き始めているのが2026年度です。SSBJ基準では「合理的な推計値の使用」も認められていますが、データ品質向上の観点から取引先からの**実測値提供への圧力は年々高まる**見通しです。

実際に大手商社・メーカーのアンケートでも、「サプライヤーへのGHGデータ提供依頼を2026年度中に開始する」と回答した企業が増加しており、準備のない中堅・中小サプライヤーは取引継続上のリスクを抱えかねない状況になっています。

## 3. 取引先から来る「排出量データ提供依頼」——現場で起きていること

すでに製造業の現場では、こうした事態が現実のものとなっています。

**よくある問い合わせの形式例：**
- 「2026年度分のGHG排出量（Scope1・2）を教えてください」
- 「当社のScope3カテゴリ1算定のため、製品1個あたりのCO2排出量を提供してください」
- 「自社のGHG排出量算定・開示の取り組み状況を、下記フォームにご回答ください」

こうした要請への対応が遅れると、取引先の評価に影響するケースも出てきています。特に自動車・電機・精密機械など、グローバルサプライチェーンに組み込まれた製造業では、欧州CSRD（企業サステナビリティ報告指令）への対応を迫られるグローバル企業からの要請も重なり、**対応の優先度が急速に上がっています。**

一方で、多くの中堅・中小製造業では、「電力使用量は把握しているが、CO2換算値がわからない」「拠点が複数あり、全体を一元管理できていない」「担当者がおらず、どこから手をつければよいかわからない」というのが実態です。

## 4. サプライヤーに求められる最低限のデータ整備——今着手すべき3つのステップ

取引先からの要請に対応できる体制を最短で整えるには、以下の3ステップが有効です。

### ステップ1：Scope1・2の排出量を拠点別に把握する

最初の一歩は、自社のScope1（燃料燃焼など直接排出）とScope2（電力使用など間接排出）を拠点・設備単位で把握すること。多くの要請はまずScope1・2から始まります。

- 電力使用量（kWh）→ 電力会社の請求書・検針票から収集
- ガス・燃料使用量（m³・L）→ 購買記録から収集
- CO2換算：排出係数（電力会社の公表値、GHGプロトコル係数）を使用

### ステップ2：算定ルールを社内で統一し、記録・保管する

「毎年同じ方法で算定・記録できる仕組み」がなければ、継続的なデータ提供はできません。

- 算定境界（どの拠点・設備を含めるか）を文書化
- 年度ごとの排出量を記録・保管（少なくとも過去3〜5年分）
- 算定根拠（使用した排出係数・データソース）を残す

### ステップ3：削減の「道筋」を示せる準備をする

データを提供するだけでなく、「削減に向けてどう取り組んでいるか」を示せると、取引先評価や入札対応での競争力が高まります。

- 主要設備の省エネ状況・更新計画
- 再エネ調達・太陽光導入の検討状況
- 削減目標（中期・長期）の設定
