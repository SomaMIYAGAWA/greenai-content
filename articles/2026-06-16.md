# 「Scope3は推計でいい」は半分だけ正しい——SSBJ開示セーフハーバー3要件を満たすために多拠点製造業が今すぐ整えるべきデータ管理体制

**メタディスクリプション：**
2026年4月施行の開示府令改正でSSBJ義務化が確定（2027年3月期〜）。Scope3にはセーフハーバー・ルールがあるが免責には3要件の文書化が必要。多拠点製造業が今すぐ取り組むべき実務ステップを解説。

---

「Scope3はとりあえず推計でも開示できる——セーフハーバー・ルールがあるから」。2027年3月期に向けたSSBJ開示対応を進める製造業の脱炭素担当者の間で、こうした安心感が広がっている。しかし、この理解は半分しか正しくない。セーフハーバー・ルールは「根拠なし推計」を免責するものではなく、**3つの要件を文書で証明できる場合にのみ**有効だ。義務化まで残り9ヶ月——多拠点製造業が今すぐ取り組むべき実務を、具体的な手順で解説する。

---

## 1. SSBJ基準の義務化タイムラインが確定——2027年3月期まであと9ヶ月

2026年4月、金融庁は「企業内容等の開示に関する内閣府令」等の改正を施行し、SSBJ（サステナビリティ基準委員会）の開示基準を有価証券報告書に組み込む法的枠組みが確定した。

適用タイムラインは以下の通りだ。

| 対象企業 | 義務化開始 |
|---|---|
| 東証プライム上場・時価総額3兆円以上 | **2027年3月期**（=今から約9ヶ月後） |
| 東証プライム上場・時価総額1兆円以上 | 2028年3月期 |
| 東証プライム上場・時価総額5,000億円以上 | 2029年3月期（予定） |

SSBJ基準はISSBのIFRS S2号（気候関連開示）の日本版であり、有価証券報告書において以下の情報開示が義務付けられる。

- **ガバナンス**：取締役会・経営陣の気候変動リスク管理体制
- **戦略**：気候シナリオ分析に基づく事業リスク・機会の評価
- **リスク管理**：GHG排出量の管理プロセス
- **指標と目標**：Scope1・2・3の排出量数値および削減目標

特に多拠点製造業にとって壁となるのが、**Scope3（バリューチェーン排出量）の開示**だ。Scope3は調達先・物流・製品使用・廃棄まで含む15カテゴリで構成され、自社データだけでは算定できない。

## 2. 「セーフハーバー・ルール」の本当の意味——3要件を満たさなければ免責されない

SSBJ基準のScope3開示には「セーフハーバー・ルール」が設けられている。開示した排出量に誤りがあっても一定条件を満たせば虚偽記載責任を問われない、という仕組みだ。

しかし、このルールは「推計すれば何でもよい」という免罪符ではない。免責を受けるためには以下の**3要件すべて**を満たした上で、その根拠を文書化しておく必要がある。

1. **第三者情報の適切性説明**：サプライヤーや業界統計など自社管理外の情報を使用した場合、その情報源が適切であることを説明できること
2. **推計前提の内部検討証明**：「なぜこの係数を使ったか」「なぜこのカテゴリを重要と判断したか」について、合理的な内部検討プロセスを記録・説明できること
3. **開示内容の合理的範囲**：業界平均や他社比較と照らして、開示した推計値が著しく乖離していないこと

つまり、**「根拠なし推計」「係数の丸投げ算定」「重要性判断の説明なし」では、セーフハーバーは機能しない**。投資家や監査人から算定根拠を問われたとき、その根拠を文書として示せるかどうかが免責の分岐点となる。

## 3. 多拠点製造業の「重要Scope3カテゴリ特定」——なぜデータ整備が先か

SSBJ基準はScope3の全15カテゴリを開示することを要求してはいない。「**重要なカテゴリを特定して開示する**」というアプローチを認めている。これは企業の負担軽減に見えるが、実は大きな落とし穴を含んでいる。

**「重要なカテゴリ」を特定するためには、各カテゴリの排出量規模を把握しなければならない。つまり、カテゴリ別の排出量データなしに、どのカテゴリが重要かを判断することは不可能だ。**

製造業でScope3排出量が大きくなりやすい主なカテゴリは以下の通りだ。

- **カテゴリ1（購入した製品・サービス）**：調達する原料・部品の製造由来の排出。製造業では全Scope3の50〜80%を占めることも多い
- **カテゴリ11（販売製品の使用）**：自社製品が顧客に使用される際の排出。特に機械・電機・自動車メーカーで重要
- **カテゴリ4（輸送・配送）**：物流委託先の輸送に伴う排出

多拠点製造業では、拠点ごとに調達先・物量が異なるため、「全社合算のカテゴリ1」を算定するだけでも、各拠点の購買データを集約する仕組みが必要になる。現場感覚として「本社が各拠点に手集めで依頼するだけで3ヶ月かかった」という事例は珍しくない。

**今すぐデータ収集を始めない企業は、「重要カテゴリを特定できなかった」という状態で2027年3月期の報告期限を迎えるリスクがある。**

## 4. 9ヶ月で整備する実務ロードマップ——フェーズ別アクションプラン

2027年3月期の有価証券報告書提出（2027年6月頃）に向けて、逆算した実務ステップは以下の通りだ。

### Phase 1（〜2026年8月）：Scope1・2の拠点別データ精度を高める
- 拠点ごとのエネルギー使用量・CO2排出量を月次で自動集計できる体制の整備
- 算定係数（電力排出係数等）の文書化
- Scope3算定の起点となる「購買データ」「輸送量データ」の棚卸し

### Phase 2（2026年9〜11月）：主要Scope3カテゴリの算定開始
- カテゴリ1（購入した製品・サービス）：主要調達先TOP20へのデータ依頼開始、一次データが取れない場合は産業連関表・排出原単位データベースを活用した推計
- カテゴリ11（販売製品の使用）：主力製品の使用段階排出係数を算出
- 上記以外のカテゴリについても規模感を把握し「重要性判断」の文書化

### Phase 3（2026年12月〜2027年2月）：開示文書の草案作成
- 重要なScope3カテゴリの確定と、その判断根拠の文書化
- セーフハーバー・ルール3要件に対応する算定根拠文書の整備
- 気候シナリオ分析・削減ロードマップとの整合確認

### Phase 4（2027年3月〜）：有価証券報告書への記載・外部レビュー
- 第三者保証（限定的保証から開始）の実施
- 有価証券報告書サステナビリティセクションへの記載
- 翌年度以降の精度向上計画の策定

SSBJ開示対応と脱炭素ロードマップ策定は、見た目は別々の作業に思えるが、根拠データは共通している。「拠点別・カテゴリ別のCO2排出量の正確な把握」が、両者の出発点だ。

## まとめ

- **セーフハーバー・ルールは「根拠なし推計」を守らない**：免責を受けるには「第三者情報の適切性説明」「推計前提の内部検討証明」「開示内容の合理的範囲」という3要件の文書化が必要
- **「重要カテゴリの特定」にはデータが先**：どのScope3カテゴリが重要かを判断するためには、カテゴリ別の排出量規模把握が不可欠。データ基盤なしの重要性判断はセーフハーバーを機能させない
- **2027年3月期まで9ヶ月——実務フェーズは今月から**：Phase1（拠点別Scope1・2精度向上）を今月中に開始しなければ、Phase2以降が後ろ倒しになり、開示書類の草案作成フェーズで間に合わなくなるリスクがある
