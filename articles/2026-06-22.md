# 工場の変圧器が「GX-ETSの隠れたリスク」になる——2026年4月施行・第三次トップランナー基準と多拠点製造業が今すぐ行うべき受変電設備棚卸の実務ガイド

**メタディスクリプション：** 2026年4月施行の変圧器トップランナー第三次基準で、旧式変圧器の無負荷損失がGX-ETS排出枠コストを24時間押し上げるリスクが顕在化。稼働中の57%が旧式機種という現実を受け、多拠点製造業の脱炭素担当者が今すぐ取り組むべき受変電設備棚卸・更新戦略を実務ガイドとして解説する。

---

「省エネ投資の優先リストに変圧器が入っていない」——脱炭素担当者と話すと、こうした声を聞くことが少なくない。LEDへの切り替えや空調の高効率機への更新、工場ボイラーの燃料転換は具体的な投資案件として俎上に上がりやすい。しかし変圧器（受変電設備）は「電気が来ている間は動いている」という性質上、存在を忘れられがちな設備だ。

2026年4月1日、その変圧器を巡る状況が大きく変わった。省エネ法に基づくトップランナー制度の**第三次判断基準**が施行され、旧式変圧器のエネルギー効率問題がGX-ETS（排出量取引制度）の財務コストと直結する構造が整ったからだ。

---

## 変圧器の「無負荷損失」とは何か——なぜ工場の電力を24時間無駄に消費し続けるのか

変圧器には2種類の損失がある。**負荷損失**（電流が流れるときに発生する損失）と、**無負荷損失**（電圧をかけている限り、負荷の有無に関わらず常時発生する鉄心の損失）だ。後者は「コア損失」「鉄損」とも呼ばれ、工場が稼働していない深夜・休日でも止まることなく電力を消費し続ける。

工場の受変電設備には、特別高圧を高圧へ落とすトランス、各棟・各ライン向けの低圧トランスなど、多拠点製造業では**1拠点あたり10〜30台以上**の変圧器が設置されているケースが多い。100kVA〜500kVA級の汎用変圧器では、旧型機種で年間**2,000〜8,000kWh**の無負荷損失が発生する試算がある。

この数字を単体で見ると「大したことない」と感じるかもしれない。しかしGX-ETSが本格稼働した2026年度以降、この損失は次節で示す通り「排出枠コスト」として経営に跳ね返ってくる。

## 2026年4月施行・第三次トップランナー基準の衝撃——日本の変圧器57%が「20年超の旧式」という現実

省エネ法のトップランナー制度は、省エネ基準を超える高効率製品の普及を促す仕組みで、変圧器については2005年（第一次）、2014年（第二次）に続き、**2026年度が第三次判断基準の目標年度**となった。

JEMA（日本電機工業会）の試算では、日本国内で稼働する変圧器は約**386万台**（油入変圧器：351万台、モールド変圧器：35万台）。そのうち製造から20年以上が経過した**2001年度以前の機種が約221万台、全体の57%**を占める。これらは第一次基準すら満たさない旧式機種であり、現行の第三次基準と比較すると無負荷損失が大幅に大きい。

新基準の省エネ効果は以下の通りだ。

| 比較対象 | 無負荷損失の削減率 |
|---|---|
| JIS旧規格（1981年基準）比 | 約46%削減 |
| 第一次トップランナー基準比 | 約26%削減 |
| 第二次トップランナー基準比 | 約11.4%削減 |

重要なのは、**2026年4月1日以降は旧基準品の出荷が禁止**される一方、既設の旧式変圧器を「継続使用すること」は制度上禁止されていない点だ。更新を義務付けられていないがゆえに、担当者が判断を先送りにしやすい。しかし次節で述べるGX-ETSの観点から見ると、この「先送り」は毎年コストを積み上げていることになる。

## GX-ETS×変圧器損失——「排出枠のコスト」で試算すると何が変わるか

2026年4月から本格稼働したGX-ETSでは、CO2直接排出量が年平均10万t以上の事業者に排出枠の保有義務が生じる（間接排出・Scope2も排出目標の対象）。排出枠の炭素価格は現在**上限4,300円/t-CO2**で設定されており、排出量が枠を超過すると追加購入が必要になる。

以下の試算例で考えてみよう。

**【モデルケース：製造業工場1拠点、旧式変圧器20台保有】**

| 条件 | 数値 |
|---|---|
| 旧式変圧器（200kVA級）1台の無負荷損失 | 年間 約3,000kWh |
| 変圧器台数 | 20台 |
| 拠点全体の無負荷損失 | 年間 60,000kWh |
| CO2換算（電力排出係数0.45kg/kWh） | 約27t-CO2/年 |
| GX-ETS排出枠コスト（4,300円/t） | 約11.6万円/年 |
| 第三次基準品への更新で削減（-26%） | 約7t-CO2削減 |
| 排出枠コスト削減効果 | 約3万円/年 |

金額だけで見ると「年間3万円」と小さく映るかもしれない。しかし**多拠点展開している製造業では拠点数×設置台数の掛け算**になる。5拠点×20台=100台の規模なら、GX-ETS絡みの排出枠コスト削減は**年間約15万円以上**に相当し、省エネ補助金（後述）を活用した更新コストを5〜10年で回収できる計算になる。

加えて、変圧器を更新しないまま排出目標を達成しようとすれば、他の設備投資や再エネ調達でそのギャップを埋める必要があり、結局そちらのコストが増す。まず「足元の損失を削る」発想で変圧器を見直すことが、脱炭素ロードマップの効率化につながる。

## 多拠点製造業の実務ステップ——変圧器棚卸から更新計画策定まで4つのフェーズ

変圧器更新プロジェクトを進める際には、以下の4フェーズで体系的に取り組むことを推奨する。

### フェーズ1：受変電設備台帳の整備（棚卸）
まず各拠点の変圧器について、**①製造年、②定格容量（kVA）、③型番・規格（旧JIS/第一次/第二次/第三次）、④設置場所**を一覧化する。設置から20年以上経過している機種を「優先更新候補」として分類する。

### フェーズ2：無負荷損失の定量化
カタログや銘板データから無負荷損失（W）を確認し、年間稼働時間（8,760時間）との積でkWhを算出する。これをCO2換算（電力排出係数を使用）してScope2排出量への寄与分を可視化する。

### フェーズ3：更新ROIの計算と優先順位の決定
更新コスト（機器費+工事費）を、電力コスト削減効果（kWh削減×電力単価）+ GX-ETS排出枠節約効果で割り戻し、投資回収年数を拠点別・機種別に試算する。稼働率が高く、容量の大きい変圧器が交換優先度の高い候補になりやすい。

### フェーズ4：補助金申請とGX-ETS移行計画への組み込み
更新計画を**GX-ETS移行計画**（2026年9月30日提出期限）の「Scope2削減施策」として明示的に記載する。省エネ・非化石転換補助金では高効率変圧器が補助対象になるケースがあり、初期コストの圧縮が可能だ（次節参照）。

## 省エネ補助金の活用と更新後の効果管理——PDCAに組み込む方法

省エネ・非化石転換補助金（経産省・SII）では、高効率変圧器への更新が「電気需要最適化設備」として補助対象になり得る。補助率は**中小企業1/2、大企業1/3**が基本だ（公募要件・年度により異なるため最新情報を要確認）。また、環境省のSHIFT事業（脱炭素技術等による工場・事業場の省CO2化加速事業）でも、変圧器更新を含む省エネ設備導入を支援する補助メニューがある。

更新後は、変圧器の**無負荷損失削減量をScope2排出量の削減実績として記録・報告**することが重要だ。GX-ETS移行計画や省エネ法定期報告に反映するためには、更新前後の損失データを比較可能な形でシステムに記録しておく必要がある。特に多拠点製造業では、どの拠点のどの変圧器を何年に更新し、削減効果がいくらだったかを一元管理する体制が求められる。
